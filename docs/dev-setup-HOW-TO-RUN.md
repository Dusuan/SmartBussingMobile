# 🚍 Smart Bussing

Aplicación para visualizar rutas de autobuses y optimizar trayectos de un punto A a un punto B.

## 🚀 Setup del proyecto

Sigue estos pasos para comenzar a desarrollar:

### 1. Instalar dependencias

```bash
npm install
```

### 2. Generar módulos nativos

```bash
npx expo prebuild
```

⚠️ Esto generará las carpetas de iOS y Android.
NO las edites manualmente.

### 3. Ejecutar la aplicación

Para iOS:

```bash
npx expo run:ios
```

Para Android:

```bash
npx expo run:android
```

## 🧠 Notas

* Este proyecto usa Expo con development builds (no Expo Go).
* Asegúrate de tener configurado Android Studio o Xcode.
